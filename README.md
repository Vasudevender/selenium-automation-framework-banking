# 🚀 Selenium Automation Framework - Banking Domain

## 📌 Project Overview
This project demonstrates a scalable and maintainable **Selenium WebDriver Automation Framework** built using Java, TestNG, and Maven.

It follows the **Page Object Model (POM)** design pattern to improve code reusability, readability, and maintainability.

---

## 🛠 Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven
- POM (Page Object Model)
- Git

---

## 📂 Framework Features
✔ Scalable automation framework design  
✔ Page Object Model implementation  
✔ Cross-browser testing support  
✔ TestNG reporting & parallel execution  
✔ Data-driven testing support
